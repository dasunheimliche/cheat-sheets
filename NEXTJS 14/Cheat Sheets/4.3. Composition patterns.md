## F - Pasando Props de Server Components a Client Components (Serializaci√≥n)

En este ejemplo, `posts` (un array de objetos con strings) es serializable y se puede pasar sin problemas al Client Component `PostList`.

#### 5. **Notas o advertencias:**

- **Serializaci√≥n es clave**: Aseg√∫rate de que los props que pasas de Server a Client Components sean serializables.
