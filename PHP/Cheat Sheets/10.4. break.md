## A - Concepto: break

**Definición:** La estructura break en PHP termina la ejecución de la estructura de control actual, como for, foreach, while, do-while o switch. Puede aceptar un argumento numérico opcional que indica cuántas estructuras anidadas deben ser terminadas.

**Ejemplo:**

```php
$arr = array('one', 'two', 'three', 'four', 'stop', 'five');
foreach ($arr as $val) {
    if ($val == 'stop') {
        break; // También se podría escribir 'break 1;'
    }
    echo "$val<br />\n";
}
```

## B - Concepto: Argumento numérico

**Definición:** El argumento numérico opcional en break especifica cuántas estructuras anidadas deben ser terminadas. El valor predeterminado es 1, lo que significa que solo se termina la estructura de control inmediata.

**Ejemplo:**

```php
$i = 0;
while (++$i) {
    switch ($i) {
        case 5:
            echo "At 5<br />\n";
            break 1; // Salir solo del switch
        case 10:
            echo "At 10; quitting<br />\n";
            break 2; // Salir del switch y del while
        default:
            break;
    }
}
```

## C - Concepto: Uso en bucles

**Definición:** break se utiliza comúnmente en bucles (for, foreach, while, do-while) para terminar la ejecución del bucle antes de que se complete todas las iteraciones.

**Ejemplo:**

```php
for ($i = 0; $i < 10; $i++) {
    if ($i == 5) {
        break;
    }
    echo $i;
}
```

## D - Concepto: Uso en switch

**Definición:** En una estructura switch, break se utiliza para terminar la ejecución del caso actual y evitar que el código continúe ejecutándose en los casos siguientes.

**Ejemplo:**

```php
$color = "rojo";
switch ($color) {
    case "rojo":
        echo "El color es rojo";
        break;
    case "azul":
        echo "El color es azul";
        break;
    default:
        echo "El color no es rojo ni azul";
}
```
