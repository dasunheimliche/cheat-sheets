## A - Operador de unión (+)

**Definición:** El operador de unión (+) combina dos arrays. Si hay claves duplicadas, se mantienen los valores del primer array.

**Ejemplo:**

```php
$a = array("a" => "apple", "b" => "banana");
$b = array("a" => "pear", "b" => "strawberry", "c" => "cherry");
$c = $a + $b; // Unión de $a y $b
print_r($c);
```

Resultado:

```php
Array
(
    [a] => apple
    [b] => banana
    [c] => cherry
)
```

En este ejemplo, los valores de $a se mantienen para las claves duplicadas.

## B - Operador de igualdad (==)

**Definición:** El operador de igualdad (==) devuelve true si dos arrays tienen los mismos pares clave/valor, sin importar el orden.

**Ejemplo:**

```php
$a = array("apple", "banana");
$b = array(1 => "banana", 0 => "apple");
var_dump($a == $b); // bool(true)
```

En este ejemplo, $a y $b son iguales porque tienen los mismos pares clave/valor.

## C - Operador de identidad (===)

**Definición:** El operador de identidad (===) devuelve true si dos arrays tienen los mismos pares clave/valor en el mismo orden y del mismo tipo.

**Ejemplo:**

```php
$a = array("apple", "banana");
$b = array(1 => "banana", 0 => "apple");
var_dump($a === $b); // bool(false)
```

En este ejemplo, $a y $b no son idénticos porque el orden de los elementos es diferente.

## D - Operador de desigualdad (!=)

**Definición:** El operador de desigualdad (!=) devuelve true si dos arrays no son iguales.

**Ejemplo:**

```php
$a = array("apple", "banana");
$b = array("apple", "cherry");
var_dump($a != $b); // bool(true)
```

En este ejemplo, $a y $b no son iguales porque tienen diferentes valores.

## E - Operador de desigualdad alternativo (<>)

**Definición:** El operador de desigualdad alternativo (<>) también devuelve true si dos arrays no son iguales.

**Ejemplo:**

```php
$a = array("apple", "banana");
$b = array("apple", "cherry");
var_dump($a <> $b); // bool(true)
```

En este ejemplo, $a y $b no son iguales porque tienen diferentes valores.

## F - Operador de no identidad (!==)

**Definición:** El operador de no identidad (!==) devuelve true si dos arrays no son idénticos.

**Ejemplo:**

```php
$a = array("apple", "banana");
$b = array(1 => "banana", 0 => "apple");
var_dump($a !== $b); // bool(true)
```

En este ejemplo, $a y $b no son idénticos porque el orden de los elementos es diferente.
